# Reference Docs

Generieke referentiedocumenten voor het IMS-platform. Dit zijn niveau 1- en niveau 2-documenten uit de IMS-documentenpyramide die als template of blauwdruk dienen — niet organisatiespecifiek ingevuld.

## Documentenpyramide

```
Niveau 1: IMS Handboek          — bestuurlijk kader, governance, scope, PDCA-structuur
Niveau 2: Domeinplannen         — BIO-implementatieplan, AVG-procedures, BCP's
Niveau 3: Procedures/werkinstr. — concrete werkwijzen, checklists, templates
Niveau 4: Registraties/bewijs   — risicoregister, SoA, auditlogboeken, GRC-tool
```

## Inhoud

| Bestand | Niveau | Beschrijving |
|---------|--------|--------------|
| *(zie ims-proces/blueprint-handboek.md)* | 1 | Blueprint IMS Handboek — generiek skelet met placeholders |

## Gebruik

- **Blueprint-documenten** worden door AI agents ingevuld op basis van wat een organisatie tijdens het IMS-proces invoert
- **Organisatiespecifieke invullingen** worden nooit in deze map opgeslagen — die horen in de tenant-omgeving of SharePoint van de betreffende organisatie
- Nieuwe templates toevoegen? Gebruik dezelfde `{{PLACEHOLDER}}`-syntax als het blueprint-handboek
